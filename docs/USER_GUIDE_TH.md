# MMK1000 User Guide (TH)

คู่มือผู้ใช้งานหน้าเว็บ MMK1000 (ฉบับสั้น)

## ขั้นตอนใช้งานหลัก

1. เปิดเว็บ `http://127.0.0.1:4100`
2. ใส่ `ADMIN KEY` แล้วกดปุ่มบันทึก
3. ไปที่เมนู `แดชบอร์ด` แล้วกดรีเฟรชเพื่อตรวจยอด/รายการ
4. ไปที่เมนู `ถอนเงิน` แล้วกรอกประเภท + จำนวน + ปลายทาง จากนั้นกดเพิ่มเข้าคิว
5. ในตารางคิว กด `อนุมัติ`
6. กด `ส่งโอน`
7. ตรวจผลในคิวว่ารายการเปลี่ยนสถานะ (เช่น `pending` → `approved` → `sent` หรือ `failed`)

## เตือนสำคัญ: mock vs real

- `mock` คือโหมดทดสอบ ไม่โอนเงินจริง
- `real` คือโหมดใช้งานจริง โอนเงินจริง

## วิธีเช็กว่า real ทำงานอยู่

- ดูผลตอบกลับจากระบบ: ถ้าเป็นโหมดทดสอบมักมีค่า `mock:true` หรือ `mode: "mock"`
- ถ้าเป็นโหมดจริง ต้องไม่แสดงผลลักษณะ `mock:true`
- ตรวจค่า `TMN_MODE` ในไฟล์ `.env` ให้เป็น `real` ก่อนเริ่มงานจริง

## ข้อแนะนำความปลอดภัย

- ห้ามแชร์ `ADMIN KEY`, token, pin หรือค่า secret
- ห้าม commit ไฟล์ `.env` ขึ้น Git
